Histogram

A histogram is the most basic graph type used to visualize a distribution. A crucial point is to choose a proper bin size. The bin size represents a trade-off between bias and variance. Ideally, we want a value that allows us to show the shape of the data but hides the noise. I manually chose 30 as the bin size because it seems like a good value in this case.

Boxplot

Boxplots are a way to summarize data distributions through their quartiles. It's very convenient if we want to compare distributions between groups, as it doesn't take much space. It's important to remember that we are hiding some information that might be important.

Scatter plot

Scatter plot are used to observe relationships between variables. In this case we can see the diameter is highly correlated with the absorbed CO2 . As the diameter increases the absorbed CO2 grow linearly until it suddenly drops. Probably, these are very old trees potentially diseased or dying.
