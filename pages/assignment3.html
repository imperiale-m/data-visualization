Trento tree abundance

Tree abundance per district

Choropleth map

A choropleth map is a type of thematic map in which administrative areas are colored according to a statistical range. In this case I used Jenks algorithm for the discretization of the colormap.
Jenks Wikipedia

Trento tree density

Tree density per district

Choropleth map normalization

We can normalize the data by deriving a spatially intensive variable from one or more spatially extensive variables. In this case we used the density which is equal to the ratio between the total and area.

Trento oxygen production

Oxygen production per district

Choropleth map

This is like the first map but this time we used the oxygen production to create the colormap. The two maps are similar because the oxygen production is strongly correlated with the abundance of trees, but they are not exactly the same because the tree specie also play a role for the oxygen production.

Trento tree map

Dot density map showing all trees in Trento

Dot density map

Here we are visualizing each tree in Trento as a circle in the map. We can easily see the density in each region of the map, but if we want to distinguish each tree distinctively we should implement a zoom since most trees are close to each others.
